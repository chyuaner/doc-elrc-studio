動態歌詞編輯器
===

## 一、 無縫雙向轉換的「三階歌詞工作流」
不論是從零開始，還是拿到現成的未對時文字，本系統都將歌詞製作拆解為遞進、非破壞性的三個核心操作區塊：
*   **原始文字預處理（Text Editor）**
    *   **痛點解決**：歌詞來源通常格式混亂，包含多餘的空白行、引言、多種語系。
    *   **特色功能**：
        *   **雙向繁簡轉換**：內建 lib/chinese-conv 引擎，一鍵將簡體歌詞轉換為繁體標準（或反向），避開不同播放器顯示亂碼的問題。
        *   **語意型文字清理**：自動忽略不必要的重複句空白，支援在文字編輯階段進行「段落定義」，這些定義與空白將在匯入同步時間軸時，自動被轉化為「間奏 (Intro/Interlude)」偵測點。
*   **高效率動態歌詞同步（Sync Editor）**    
    *   **痛點解決**：逐字/逐句對時是一項極耗專注度與體力的工作。傳統編輯器「拍錯一個字，後面整句得重對」。
    *   **特色功能**：
        *   **無滑鼠純鍵盤對時**：製作者只需雙手放在鍵盤上，即可透過敲擊（拍點鍵 / 快捷鍵）完成逐字、逐句的分毫對時。
*   **即時卡拉OK preview（Karaoke Preview）**
    *   **痛點解決**：製作者過去必須將歌詞導出到外部播放器，甚至進剪輯軟體壓制，才能確認實際在螢幕上的滾動節奏與排版。
    *   **特色功能**：
        *   **零延遲即時渲染**：Preview 視窗會抓取當前同步編輯器中的動態狀態（Active state），在音軌播放時，直接在網頁端模擬雙行、單行置中以及縮放排版，讓製作者「隨改隨看」。

## 二、 外科手術式的「右鍵上下文選單」編輯體驗（核心亮點）
同步編輯器（Sync Editor）中精心設計的 **右鍵上下文選單（Context Menu）**，是將此專案推向專業商用等級的關鍵。它移除了繁瑣的代碼手動輸入，允許製作者直接用滑鼠指標進行「直覺式微調」：
*   **字元級分割（Divide）與合併（Merge）**：
    *   當歌詞歌手唱腔有連音、或歌詞需要將詞組拆開為多個字（或合併多字為一個唱拍）時，製作者無需手動修改底層的時間軸代碼，在右鍵選單點擊即可瞬間重組字元節奏。
*   **高精度時間與時長微調（Fine-Grained Offset）**：
    *   支援直接透過選單對單一字元、甚至整行歌詞進行毫秒（ms）等級的時間軸平移（Shifting）或持續時間編輯。
*   **多歌手與顏色快速指派（Singer Attribution）**：
    *   右鍵選單內嵌「多歌手名單快速設置」。製作者可任意圈選一個字或整句歌詞，右鍵一鍵變更歌手（如：主唱、副唱、合唱、旁白），系統會自動在 ELRC/ASS 的底層代碼中注入對應的 kstyle 歌手自訂標記。

## 三、 底層音訊標籤內嵌技術（Audio Metadata Tagging）
這是本編輯器最接近「商業封裝軟體」的技術賣點。一般歌詞製作工具有歌詞導出，但要寫回音檔，製作者通常得再開一個專門的 Tag 編輯軟體（如 Mp3tag）。
本系統內建了**高階音訊 Metadata 解碼與編碼寫入引擎**：
*   **FLAC 與 M4A 零失真直接寫入（FLAC/M4A Utility）**：
    *   **FLAC 歌詞寫入**：利用 lib/flac-utils 解析 FLAC 檔案結構中的 Metadata block，直接將產出的 LRC 或 ELRC 歌詞注入 LYRICS 屬性中，且完全不影響無損音軌的位元流    
    *   **M4A (AAC) 封裝語音**：利用 lib/m4a-utils 解析 M4A/AAC 的 atom 容器結構，尋找 ilst 節點，正確寫入 lyr 原子標籤（Atom），達到 Apple Music 與其他主流高解析度播放器完美識別的要求。
*   **一鍵產出**：使創作者在對時完畢後，直接下載「已內嵌歌詞的專屬實體音檔」。

## 四、 工業級多格式高度相容導出（Cross-Format Exporter）

歌詞製作完成後，製作者往往需要部署在不同的播放端與應用情境。本系統的 Exporter 支援導出多種具有高相容性的標準格式：

*   **LRC (標準句級)**：支援在不具備逐字顯示功能的傳統播放設備（如車載、老舊 MP3）上播放。
*   **Enhanced LRC / ELRC (字元級動態)**：相容於現代進階播放器，支援流暢的逐字上色演出（包含歌手變色屬性）。
*   **SRT (標準影視字幕)**：適用於剪輯軟體（Premiere, DaVinci Resolve, Final Cut）的快速導入，方便製作一般 MV 影片。
*   **KTV ASS (王牌格式)**：高還原度的卡拉OK雙行滾動（含圓點倒數指示與多人分色對唱），適用於外掛字幕或直接進行大螢幕卡拉OK壓制。
