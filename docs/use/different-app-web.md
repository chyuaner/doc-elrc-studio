桌面視窗版 與 Web線上網頁版 有什麼不一樣
===

雖然桌面版與手機平板版都是共用同一個線上網頁版核心，目前因為環境不同，該環境允許能做的事情不同（APP權限不同），在功能上會有些許不同。

| 項目 | 適用平台 | 自動開啟相關檔案 | 檔案類型關聯 |
| --- | --- | :---: | :---: |
| 桌面視窗版 | Linux / macOS / Windows | ✅ | ⬜ |
| 手機平板APP版 | Android / iPadOS(iOS) | ❌ | ⬜ |
| Web線上網頁版 | Firefox / Chromium (Chrome) | ❌ | ❌ |

圖例：
- ✅ 有支援
- ❌ 不支援
- ⬜ 尚未實作
