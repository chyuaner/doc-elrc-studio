我已經為您盤點了 `web-elrc-studio` 專案中所有自訂的擴充標籤。這些標籤主要用於擴充標準 LRC 格式，以支援更複雜的 **KTV ASS 歌詞排版**、**多人對唱顏色**以及**間奏 Logo 顯示**。

本專案的標籤分為兩大類：**「全域屬性標籤」**（寫在文件開頭）與**「行內控制標籤」**（寫在歌詞行中）。

---

## 一、 全域屬性標籤 (Global Metadata Tags)
這些標籤通常位於 LRC 文件頂部，用於定義整首歌的 KTV 輸出設定。

| 標籤名稱 | 用途說明 | 用法範例 |
| :--- | :--- | :--- |
| `[kti:文字]` | **KTV 專用歌名**。用於開場資訊，可與標準 `[ti:]` 分離。 | `[kti:不能說的祕密]` |
| `[kar:文字]` | **KTV 專用歌手**。用於開場資訊，支援多行（以 `\n` 分隔）。 | `[kar:周杰倫]` |
| `[kal:文字]` | **KTV 專用專輯**。用於開場資訊。 | `[kal:依然范特西]` |
| `[ko:文字]` | **KTV 自訂資訊**。用於開場資訊的最下方（如：作詞、作曲）。 | `[ko:作詞：方文山\n作曲：周杰倫]` |
| `[tt:時間]` | **資訊顯示開始時間**。控制開場資訊與 Logo 何時淡入。 | `[tt:00:00.50]` |
| `[tte:時間]` | **資訊顯示結束時間**。控制開場資訊何時淡出。 | `[tte:00:06.00]` |
| `[kth:秒數]` | **間奏閾值**。兩句間隔超過此秒數時，自動視為新段落並顯示倒數小圓。 | `[kth:6]` (預設為 6 秒) |
| `[klg:SVG]` | **間奏 Logo 圖檔**。存放 SVG 的 XML 原始碼，匯出時轉為 ASS 向量。 | `[klg:<svg.../svg>]` |
| `[klgbm:布林]` | **Logo 單色化**。是否將 Logo 強制改為單一顏色。 | `[klgbm:true]` |
| `[klgbmc:HEX]` | **Logo 單色顏色**。當 `klgbm` 為 true 時採用的顏色。 | `[klgbmc:#FFFFFF]` |
| `[klgo:布林]` | **Logo 描邊**。是否為 Logo 向量圖加上外框描邊。 | `[klgo:true]` |
| `[klgow:數字]` | **Logo 描邊粗細**。單位為像素 (px)。 | `[klgow:3]` |
| `[klgoc:HEX]` | **Logo 描邊顏色**。 | `[klgoc:#000000]` |
| `[klgno:區間]` | **Logo 排除時段**。指定哪些時間不顯示 Logo（如：有過場動畫時）。 | `[klgno:01:20.00-01:30.00]` |
| `[kstyledef:ID:文字]`| **樣式前綴定義**。定義歌詞若以特定文字開頭，自動套用樣式。 | `[kstyledef:B:男：]` (歌詞有「男：」則轉藍色) |

## 二、 行內控制標籤 (In-line Control Tags)
這些標籤插在歌詞行之間或歌詞文字中，影響單行或單字的表現。

| 標籤名稱 | 用途說明 | 用法範例 |
| :--- | :--- | :--- |
| `[ktv:singleline]` | **強制單行**。該行歌詞不參與雙行交替排版，固定顯示在第二排位置。 | `[ktv:singleline]` <br> `[01:00.00]最後一句歌詞` |
| `[ktv:center]` | **強制置中**。該行歌詞在畫面上橫向居中顯示（通常用於旁白或標題）。 | `[ktv:center]` <br> `[00:10.00]（間奏）` |
| `[ktvsp:時間]` | **段落強制分割點**。強制在此時間點淡入下一段歌詞（忽略自動計算）。 | `[ktvsp:00:27.470]` |
| `[kstyle:ID]` | **行樣式切換**。變更該行（及後續行）的顏色標記。 | `[kstyle:B]` (切換為藍色) |
| `<kstyle:ID>` | **字樣式切換**。在「增強型LRC」中，切換單個字的顏色。 | `[01:00.00]我<kstyle:R>愛你` (「愛你」變紅色) |

---

## 三、 顏色樣式 ID 定義
在 `kstyle` 標籤中使用的 ID 對應關係如下：

*   **N (Normal)**: 回歸預設值（通常是白色/橘色）。
*   **B (Blue)**: 藍色標記（通常代表「男聲」或「第一人」）。
*   **R (Red)**: 紅色標記（通常代表「女聲」或「第二人」）。
*   **P (Purple)**: 紫色標記（通常代表「第三人」）。
*   **G (Green)**: 綠色標記（通常代表「合唱」）。
*   **O (Orange)**: 橘色標記（通常代表「第四人」）。
*   **T (Text/Transparent)**: 灰色標記（通常代表「旁白」或「OS」）。

---

## 整理建議
如果您要撰寫說明文件，可以將這些標籤統稱為 **"Enhanced LRC Studio Extension Tags" (ELRC 擴充標籤)**。這些標籤的設計核心是 **「向下相容」**：在一般的 LRC 播放器中，這些標籤會被視為註解或未知標籤而忽略，不影響基本歌詞顯示；但在本專案的匯出器中，它們能轉換為高度專業的 KTV ASS 特效指令。
