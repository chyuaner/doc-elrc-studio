# KTV 專屬開源擴充 LRC 標籤系統（KTV LRC Metadata Tags）的說明

本專案為了在不破壞標準 LRC/ELRC 歌詞檔案相容性的前提下，完美實現專業級製作 KTV ASS 變色字幕、多角色演唱（男女對唱/旁白/合唱）、自訂前導標題時間、間奏 Logo 動態控制等高級功能，特別擴充設計了一套專屬 KTV 擴充標籤系統（KTV LRC Metadata Tags）。

這套擴充標籤可以完美保留於 .lrc 格式內，供本編輯器在未來重複匯入時自動套用原設定。以下為本專案專用標籤的完整盤點、功能與用法指南：

## 全域屬性標籤 (Global Metadata Tags)

### 歌曲資訊與標題設定區（Song Info & Overrides）
標準 LRC 常用 `[ti:歌名]`、`[ar:歌手]`，但本編輯器在輸出為 ASS 字幕時，為了確保格式完整、不受播放器標籤混淆並作為額外的 KTV 專用覆寫屬性，特別設計了以下 KTV 專屬標籤。只有在設定了這些專用屬性時，系統才會在 ASS 標題卡中讀取並呈現：

| 標籤名稱 | 用途說明 | 用法範例 |
| :--- | :--- | :--- |
| `[kti:文字]` | **KTV 專用歌名**。用於開場資訊，可與標準 `[ti:]` 分離。 | `[kti:不能說的祕密]` |
| `[kar:文字]` | **KTV 專用歌手**。用於開場資訊，支援多行（以 `\n` 分隔）。 | `[kar:周杰倫]` |
| `[kal:文字]` | **KTV 專用專輯**。用於開場資訊。 | `[kal:依然范特西]` |
| `[ko:文字]` | **KTV 自訂資訊**。用於開場資訊的最下方（如：作詞、作曲）。 | `[ko:作詞：方文山\n作曲：周杰倫]` |
| `[tt:時間]` | **資訊顯示開始時間**。控制開場資訊與 Logo 何時淡入。 | `[tt:00:00.50]` |
| `[tte:時間]` | **資訊顯示結束時間**。控制開場資訊何時淡出。 | `[tte:00:06.00]` |

---

### 間奏 Logo 動態與防遮蔽控制標籤（Logo Settings）
這組標籤控制在歌曲前奏、間奏或尾奏期間，畫面中央顯示的自訂 Logo 圖檔及其樣式，並支援排除時段（避免遮擋畫面或與其他字幕衝突）：

| 標籤名稱 | 用途說明 | 用法範例 |
| :--- | :--- | :--- |
| `[klg:SVG]` | **間奏 Logo 圖檔**。存放 SVG 的 XML 原始碼，匯出時轉為 ASS 向量。 | `[klg:<svg.../svg>]` |
| `[klgbm:布林]` | **Logo 單色化**。是否將 Logo 強制改為單一顏色。 | `[klgbm:true]` |
| `[klgbmc:HEX]` | **Logo 單色顏色**。當 `klgbm` 為 true 時採用的顏色。 | `[klgbmc:#FFFFFF]` |
| `[klgo:布林]` | **Logo 描邊**。是否為 Logo 向量圖加上外框描邊。 | `[klgo:true]` |
| `[klgow:數字]` | **Logo 描邊粗細**。單位為像素 (px)。 | `[klgow:3]` |
| `[klgoc:HEX]` | **Logo 描邊顏色**。 | `[klgoc:#000000]` |
| `[klgno:區間]` | **Logo 排除時段**。指定哪些時間不顯示 Logo（如：有過場動畫時）。 | `[klgno:01:20.00-01:30.00]` |


### 間奏閥值
兩句間隔超過此秒數時，自動視為新段落並顯示倒數小圓。

| 標籤名稱 | 用途說明 | 用法範例 |
| :--- | :--- | :--- |
| `[kth:秒數]` | **間奏閾值**。兩句間隔超過此秒數時，自動視為新段落並顯示倒數小圓。 | `[kth:6]` (預設為 6 秒) |

## 歌詞行內控制標籤 (In-line Control Tags)

### 該句的設定與定位標籤（Line-level Alignment & Timing）
這組標籤為單行歌詞專屬，必須緊貼在該行時間軸（如 `[01:23.45]`）之後或歌詞的前方，用以控制該行在 ASS 渲染時的版面排版行為：

| 標籤語法 | 功能名稱 | 說明與用途 | 用法與輸出結構 |
| :--- | :--- | :--- | :--- |
| `[ktv:singleline]` | 強制單行排版 | 強制該行歌詞獨立為一個段落。在雙行對唱/滾動字幕中，這行將不會與上下行合併，而是單獨上下場，適合用於前奏、間奏後的起唱句。 | **結構**：在 .lrc 檔案中將以獨立一行輸出：<br>`[ktv:singleline]` |
| `[ktv:center]` | 強制該句置中 | 讓這句歌詞在畫面上呈現「置中對齊」樣式，而非傳統 KTV 偏左/偏右的雙行排版。適用於旁白、合唱或特色過渡句。 | **結構**：在 .lrc 檔案中將以獨立一行置於歌詞上：<br>`[ktv:center]` |
| `[ktvsp:鐘點時間]` | 特殊段落起點時間 | 強制重寫此段落進場的時間點（即該行字幕「提早進場預備」的時間點，格式為 `分:秒.毫秒`），不再受制於預設的「間奏閥值自動 advance 時間」，適用於轉調、無長間奏但需提前載入字幕的特殊曲風。 | **結構**：在 .lrc 檔案中將以獨立一行置於歌詞上方：<br>`[ktvsp:00:27.470]` |
