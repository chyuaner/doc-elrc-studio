# KTV 專屬開源擴充 LRC 標籤系統（KTV LRC Metadata Tags）技術文件

本專案為了在不破壞標準 LRC/ELRC 歌詞檔案相容性的前提下，完美實現專業級製作 KTV ASS 變色字幕、多角色演唱（男女對唱/旁白/合唱）、自訂前導標題時間、間奏 Logo 動態控制等高級功能，特別擴充設計了一套專屬 KTV 擴充標籤系統（KTV LRC Metadata Tags）。

這套擴充標籤可以完美保留於 .lrc 格式內，供本編輯器在未來重複匯入時自動套用原設定。以下為本專案專用標籤的完整盤點、功能與用法指南：

---

## 一、 歌曲資訊與標題設定區（Song Info & Overrides）
標準 LRC 常用 `[ti:歌名]`、`[ar:歌手]`，但本編輯器在輸出為 ASS 字幕時，為了確保格式完整、不受播放器標籤混淆並作為額外的 KTV 專用覆寫屬性，特別設計了以下 KTV 專屬標籤。只有在設定了這些專用屬性時，系統才會在 ASS 標題卡中讀取並呈現：

| 標籤語法 | 對應設定項目 | 說明與用途 | 用法範例 |
| :--- | :--- | :--- | :--- |
| `[kti:歌名]` | KTV 專用歌名 | 專屬於 KTV 輸出時呈現在開場標題卡中的歌名。 | `[kti:海闊天空]` |
| `[kar:歌手]` | KTV 專用歌手 | 專屬於 KTV 輸出時呈現在開場標題卡中的歌手名稱。 | `[kar:Beyond]` |
| `[ko:其它]` | 欄位自訂資訊 | 其它額外的標題卡自訂揭露資訊（如作詞、作曲、編曲、製作人等）。 | `[ko:詞/曲：黃家駒]` |

---

## 二、 間奏 Logo 動態與防遮蔽控制標籤（Logo Settings）
這組標籤控制在歌曲前奏、間奏或尾奏期間，畫面中央顯示的自訂 Logo 圖檔及其樣式，並支援排除時段（避免遮擋畫面或與其他字幕衝突）：

| 標籤語法 | 對應設定項目 | 說明與用途 | 用法範例 / 參數 |
| :--- | :--- | :--- | :--- |
| `[klg:檔案名稱]` | Logo 圖檔路徑 | 儲存使用者上傳或設定的 Logo 圖片檔名或路徑。 | `[klg:my_studio_logo.png]` |
| `[klgbm:true/false]` | Logo 單色化 | 是否將 Logo 圖檔單色化（填色），常用於轉為乾淨的黑白或指定色 Logo。 | `[klgbm:true]` |
| `[klgbmc:#HEX]` | 單色化顏色 | 當 Logo 單色化啟用時，指定 Logo 本體所套用的 HEX 顏色。 | `[klgbmc:#FF5500]` |
| `[klgo:true/false]` | 啟用 Logo 外框 | 是否為 Logo 的外輪廓加上一層描邊外框。 | `[klgo:true]` |
| `[klgow:寬度]` | Logo 外框寬度 | Logo 描邊外框的寬度（像素）。 | `[klgow:3.5]` |
| `[klgoc:#HEX]` | Logo 外框顏色 | Logo 描邊外框所使用的 HEX 顏色。 | `[klgoc:#000000]` |
| `[klgno:排除時段]` | Logo 排除不顯示時段 | 規定哪些時間區段**絕對不顯示 Logo**（多個時段以逗號 `,` 分割，時間格式為 `秒.毫秒-秒.毫秒`）。第一段代表開場資訊顯示區，後續則為指定的排除區間。 | `[klgno:0.0-15.5,120.3-155.0]` |

---

## 三、 行級版面設定與定位標籤（Line-level Alignment & Timing）
這組標籤為單行歌詞專屬，必須緊貼在該行時間軸（如 `[01:23.45]`）之後或歌詞的前方，用以控制該行在 ASS 渲染時的版面排版行為：

| 標籤語法 | 功能名稱 | 說明與用途 | 用法與輸出結構 |
| :--- | :--- | :--- | :--- |
| `[ktv:singleline]` | 強制單行排版 | 強制該行歌詞獨立為一個段落。在雙行對唱/滾動字幕中，這行將不會與上下行合併，而是單獨上下場，適合用於前奏、間奏後的起唱句。 | **結構**：在 .lrc 檔案中將以獨立一行輸出：<br>`[ktv:singleline]` |
| `[ktv:center]` | 強制該句置中 | 讓這句歌詞在畫面上呈現「置中對齊」樣式，而非傳統 KTV 偏左/偏右的雙行排版。適用於旁白、合唱或特色過渡句。 | **結構**：在 .lrc 檔案中將以獨立一行置於歌詞上：<br>`[ktv:center]` |
| `[ktvsp:鐘點時間]` | 特殊段落起點時間 | 強制重寫此段落進場的時間點（即該行字幕「提早進場預備」的時間點，格式為 `分:秒.毫秒`），不再受制於預設的「間奏閥值自動 advance 時間」，適用於轉調、無長間奏但需提前載入字幕的特殊曲風。 | **結構**：在 .lrc 檔案中將以獨立一行置於歌詞上方：<br>`[ktvsp:00:27.470]` |

## 四、 多角色演唱與色彩樣式控制（Singer Styles）
KTV 字幕最靈魂的對唱功能，本專案提供高達六種預設的演唱角色（男、女、合唱、旁白等），並支援對每一種角色在 LRC 檔頭宣告自訂的標題前置代碼：

### 1. 演唱角色定義標籤（檔頭專用）
| 標籤語法 | 對應演唱角色代碼 | 預設值 | 說明與用途 |
| :--- | :--- | :--- | :--- |
| `[kstyledef:B:定義值]` | 男聲（Boy / Blue） | `男：` | 定義男聲演唱時，字幕前套用的前綴字。 |
| `[kstyledef:R:定義值]` | 女聲（Girl / Red） | `女：` | 定義女聲演唱時，字幕前套用的前綴字。 |
| `[kstyledef:G:定義值]` | 合唱（Group / Green） | `合：` | 定義合唱演唱時，字幕前套用的前綴字。 |
| `[kstyledef:P:定義值]` | 女二 / 粉紅（Pink） | *(空白)* | 定義此備用對唱角色演唱時，字幕前套用的前綴字。 |
| `[kstyledef:O:定義值]` | 其他（Other / Orange） | *(空白)* | 定義其它協力人員演唱時，字幕前套用的前綴字。 |
| `[kstyledef:T:定義值]` | 旁白 / 翻譯（Transcript | `旁白：` | 定義旁白或特殊翻譯發聲時，字幕前套用的前綴字。 |

*註：如果某個角色無須加上前綴，只需在標籤內將定義值保留為空即可（例如 `[kstyledef:P:]`）。*

### 2. 歌詞行套用角色（歌詞行專用）
| 標籤語法 | 說明與用途 | 歌詞行實際用法範例 |
| :--- | :--- | :--- |
| `[kstyle:角色代碼]` | 放在該行歌詞最前方，讓語音編譯器或隨後的 ASS 渲染器，將此行自動套用對應角色的變色字體與前置前字。可用代碼包括 **B**、**R**、**G**、**P**、**O**、**T**。 | `[kstyle:B] <00:12.30> 輕<00:12.50> 輕<00:12.80> 聽` <br>*(渲染時會自動在前方補上「男：」並套用男聲專用變色樣式)* |

---

### 設計優勢與相容性說明
1. **完全開源與向下相容**：這套擴充系統的所有標籤，皆符合標準 LRC 標籤的語法結構（以中括號 `[]` 包覆）。在一般播放器或傳統 LRC 編輯器中，不支援的播放器會自動將其當作未知標籤忽略，完全不影響標準歌詞動態播放。
2. **免除二次重複設定之苦**：使用者在本編輯器中辛苦調整的對唱設定、Logo 位置、排除時間點、置中/單行配置等，都將自動與歌詞檔（.lrc）綁定儲存。下次再次匯入同一個 .lrc 檔案，即可 100% 完美還原所有的 KTV 輸出配置！
